Описание
Проект "Графический конвертор" — это консольное приложение на C++, предназначенное для конвертации изображений между форматами JPEG, BMP и PPM. 
Программа разработана для тренировки работы с библиотеками для обработки графических файлов, таких как libjpeg для JPEG,
а также для работы с собственными реализациями для BMP и PPM.

Основная функциональность включает:

Конвертацию изображений между форматами JPEG, BMP и PPM.
Определение формата изображения по расширению файла.
Гибкий интерфейс для добавления новых форматов.

Стек технологий:
C++17
CMake (для сборки проекта)
Библиотека libjpeg для работы с JPEG
Собственные реализации для BMP и PPM
Инструкция по развёртыванию

Системные требования:
Компилятор: C++ с поддержкой C++17 (например, GCC 7.0+)
CMake: версии 3.11 или выше
libjpeg: установите её перед сборкой

Пример использования:
Программа конвертирует изображение, определяя форматы входного и выходного файлов по расширению, и выполняет конвертацию между поддерживаемыми форматами.

Планы по доработке
Добавить поддержку PNG с использованием libpng.
Улучшить обработку ошибок для более точной диагностики проблем с файлами.
Добавить многоуровневую поддержку форматов с различными цветовыми пространствами и метаданными.

Сборка

 Сборка проекта "Графический конвертор"

Для сборки проекта используется CMake, что упрощает настройку зависимостей и компиляцию. Следуйте инструкциям ниже.

 1. Установка необходимых инструментов

Убедитесь, что у вас установлены следующие инструменты:
- CMake версии 3.11 или выше
- Компилятор C++ с поддержкой C++17 (например, GCC или Clang)
- libjpeg для работы с JPEG-форматом изображений

 2. Склонируйте или скачайте проект

git clone <repository_url>
cd <repository_folder>

 3. Создание папки для сборки

Чтобы не смешивать исходные файлы с скомпилированными, создайте отдельную папку для сборки:

mkdir build
cd build

 4. Генерация Makefile с помощью CMake

Перед запуском CMake убедитесь, что указали путь к библиотеке `libjpeg`. Если путь не указан, проект не соберется, и появится ошибка: **"Please, specify LibJPEG directory via -DLIBJPEG_DIR=<dir>"**.

Для указания пути выполните следующую команду, где `<dir>` — это путь к `libjpeg`:

cmake -DLIBJPEG_DIR=<path_to_libjpeg_directory> ..


Пример для стандартной директории `/usr`:
cmake -DLIBJPEG_DIR=/usr ..

Если вы не укажете путь к `libjpeg`, сборка завершится с ошибкой:

CMake Error: Please, specify LibJPEG directory via -DLIBJPEG_DIR=<dir>


 5. Сборка проекта

После успешной генерации файлов сборки выполните команду для компиляции:

make

 6. Запуск программы

После сборки запустите программу, указав входной и выходной файлы для конвертации:

./graphic_converter ../input.jpg ../output.bmp

 Важные моменты:
- Программа поддерживает форматы JPEG, BMP и PPM.
- Убедитесь, что путь к `libjpeg` корректно передан через флаг `-DLIBJPEG_DIR`.
